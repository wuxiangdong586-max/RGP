# 混沌平台&APM监控部署流程

1.docker部署

上传docker.zip，解压安装docker和docker-compose

```
#解压安装包
unzip docker.zip
#对docker安装包赋权
chmod +x -R docker/
#切入docker脚本目录
cd docker/
#执行docker安装脚本
sh install.sh
```

2.中间件部署

上传middle.tgz，解压安装中间件

```
#解压安装包
tar -zxvf middle.tgz
#切入中间件安装目录
cd middle/
#修改common-config.yml共享配置
middle.chaos:机器ip
service.chaos:机器ip
#执行中间件安装脚本
sh install.sh
#检查容器状态，查看是否全部up
docker-compose ps
```

3.flink-job页面面配置

```
登陆flink-web页面：http://机器ip:18081/
点击页面左侧Submit New Job
点击页面右侧Add New，导入本地otel-apm-flink-server-1.0.jar
点击上面导入的jar包，配置：
Entry Class框：com.staryea.stream.serverAllAgg.ServerAllStreamingJob
Parallelism框：1
Program Arguments框：config.properties
最后点击Submit保存
```

4.平台服务部署

上传2.6.1.gz，解压安装平台服务

```
#解压安装包
tar -zxvf 2.6.1.tgz
#切入服务安装目录
cd 2.6.1/
#修改common-config.yml共享配置
middle.chaos:机器ip
service.chaos:机器ip
#执行服务安装脚本
sh install.sh
#检查容器状态，查看是否全部up
docker-compose ps
```
